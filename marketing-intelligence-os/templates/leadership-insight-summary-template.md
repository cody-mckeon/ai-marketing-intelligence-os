# Leadership Insight Summary

## 1. Executive Summary

Summarize the main story in 3-5 bullets. If actual performance data is missing, describe the intended leadership narrative framework instead of claiming results.

## 2. What We Measured

List the page, campaign, audience, date range, user actions, metrics, and data sources included in the summary.

## 3. What Changed or Happened

Summarize observed performance changes only when actual data is available. If data is missing, describe what this section will cover once GA4, Pendo, campaign, and/or vendor data is available.

## 4. Why It Matters

Explain the business significance in plain language, connecting the measured behavior to stakeholder goals, customer intent, campaign quality, or conversion visibility.

## 5. Key Signals

List the most important signals to report or monitor, such as traffic quality, qualified engagement, CTA intent, campaign quality, device differences, offer/category engagement, and third-party conversion visibility.

## 6. Data Caveats

Call out limitations, tracking gaps, attribution issues, third-party platform gaps, missing inputs, data quality concerns, or interpretation risks.

## 7. Recommended Actions

List practical next actions. If data is missing, identify the next data pulls, tracking validation steps, vendor confirmations, or stakeholder inputs needed.

## 8. Leadership-Ready Language

Write a short paragraph that could be used in a leadership update. Avoid unsupported claims and clearly caveat the summary if final data is not available.

## 9. Open Questions

List unresolved questions that require data, stakeholder clarification, vendor support, campaign documentation, or tracking validation.

## 10. Next Agent to Run

Recommend the next agent or workflow that should run after this leadership summary.
