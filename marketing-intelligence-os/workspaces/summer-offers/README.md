# Summer Offers Reporting Workspace

This workspace is for the Summer Offers reporting package.

Inputs are manually collected exports, screenshots, notes, and vendor reports. Outputs are generated or updated by the Marketing Intelligence OS agents.

Use this workspace before API or source connections are built. Manual exports are preferred first so the team can prove which data is useful before automating collection.

## Workflow

1. Confirm campaign brief details.
2. Collect GA4 exports.
3. Collect Pendo evidence if available.
4. Collect GTM tracking evidence.
5. Collect paid media reports if active.
6. Collect UTM documentation.
7. Request vendor data if available.
8. Add manual analyst notes.
9. Run Data Source Mapping Agent.
10. Run Tracking Health Agent.
11. Run Campaign Reporting Agent.
12. Run Leadership Insight Agent.
