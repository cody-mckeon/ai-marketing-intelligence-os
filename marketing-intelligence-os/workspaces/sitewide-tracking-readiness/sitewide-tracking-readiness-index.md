# Sitewide Tracking Readiness Index

## Purpose

This index tracks measurement readiness across major RWLV website page types and conversion paths.

The goal is to identify which page families are ready for reporting, which are directional only, which are not ready, and which require GTM, GA4, or vendor fixes.

## Readiness Levels

- Ready
- Ready with caveats
- Partially ready
- Not ready
- Vendor dependency
