# Summer Offers Page Performance Plan Test

## Test Name

Summer Offers Page Performance Plan Test

## Test Prompt

Using the approved Summer Offers Analytics Request Brief, create a Page Performance Analysis Plan for the Summer Offers page.

The page launched in March and includes hotel, dining, and entertainment offers. The stakeholder wants a leadership-ready understanding of traffic, engagement, CTA clicks, campaign performance, and booking intent.

## Expected Output Characteristics

The Page Performance Agent should produce a structured Page Performance Analysis Plan that defines how the Summer Offers page should be evaluated before claiming any performance results.

The plan should identify the page as an offer/campaign landing page.

The page purpose should be defined as driving qualified engagement and intent across hotel, dining, and entertainment offers.

Primary user actions should include:

- Offer engagement
- Click offer card
- Click Book Now
- Click Check Rates
- Click Reserve
- Click Buy Tickets
- Scroll to key content
- Click outbound/vendor links
- Start booking flow
- Start reservation flow
- Start ticketing flow

Measurement questions should cover:

- Traffic quality
- Engagement behavior
- Offer/category interest
- CTA behavior
- Booking, reservation, and ticketing intent
- Campaign performance
- Device behavior

Recommended data sources should include:

- GA4
- Pendo
- GTM
- Booking engine
- SevenRooms
- AXS / Fevo
- Paid media reports
- UTM documentation

The plan should separate intent clicks from completed bookings, reservations, and ticket purchases.

Data caveats should call out third-party visibility gaps.

Missing inputs should include:

- Final page URL
- Reporting date range
- CTA/event definitions
- Event names
- UTM documentation
- Vendor data availability

The plan should recommend a final page performance report structure.

The next agent recommendation should be:

- Leadership Insight Agent after page performance analysis is complete

## Acceptance Criteria

- Output uses the full 10-section Page Performance Analysis Plan format.
- Output does not claim actual performance results without data.
- Output clearly separates measurement planning from analysis conclusions.
- Output includes traffic, engagement, CTA, offer/category, campaign, device, and intent measurement.
- Output includes data caveats and tracking validation needs.
- Output recommends Leadership Insight Agent next.
